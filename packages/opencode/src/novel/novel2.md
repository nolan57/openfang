## World Building Settings

### Historical Background

现代日本都市，平凡的生活表象下隐藏着复杂的人际关系网。社会秩序井然，法律体系完善，但也因此让犯罪者无处可逃。

### Social Structure

- 市井小民：便当店员工、高中教师、无业游民
- 知识精英：大学教授、物理学家、数学家
- 执法者：刑警、司法系统

### Power System

本故事无超自然力量体系，核心"力量"是：
- 数学逻辑：石神用来构建完美犯罪的工具
- 物理推理：汤川用来破解谜题的武器
- 人性良知：最终摧毁完美逻辑的力量

### Key Locations

- 便当店：靖子工作的地方，日常生活的象征
- 出租屋：案件发生的地点，压抑的氛围
- 帝都大学：汤川和石神精神交汇的地方
- 阴暗巷弄：城市底层生活的隐喻

### Core Conflict

逻辑完美 vs 人性不完美
- 石神用极致的逻辑构建了完美的保护
- 但人性中的良知最终让这个逻辑体系崩塌

### Visual Imagery

- 数学公式/黑板：理性的象征
- 时钟/齿轮：精密计划的隐喻
- 分界线：生与死、理性与感性的边界
